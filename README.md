## 本书的读者

《Go安全编码实践指南》的主要读者是开发人员。
特别是那些以前有其他编程语言经验的人。

本书对那些第一次学习编程的人来说也是一个很好的参考。
本书对那些已经完成了[Go之旅][8]的第一次学习编程的人来说也是很好的参考。

## 你将学到什么

本书涵盖了[OWASP安全编码实践指南][3]的每个主题。
提供使用Go的例子和建议，以帮助开发人员避免常见的
错误和误区。

读完本书后，你会对自己开发安全的Go
应用程序。
